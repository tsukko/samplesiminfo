# 作った背景
- Google Play Store.へアプリをリリースする際の手順や入力項目、画面情報などについて、実際にやってみないとわからない箇所があった。
- それなりの規模の企業さんとAndroidアプリ開発の仕事をしたとき、公式情報だけでは、お客さんの理解を得ることが難しいところがあった。
これらを解決するために、作成しました。

- その他、Javaでリリース後、Kotlinに変換してリリースも試してみました。

# 機能
SIM情報にアクセスして表示するだけです。
